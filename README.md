# Todo

Netlify: carolinas-todo-app.netlify.app

Project Description
A sleek, responsive To-Do Application built with React and Vite, designed for a seamless user experience across mobile and desktop devices. The app allows users to manage their daily tasks efficiently with a clean, modern interface powered by Styled Components.Key FeaturesDynamic State Management: Utilizes Zustand for centralized global state, ensuring efficient task updates and real-time tracking of completed items.Responsive Design: Features a mobile-first approach with custom media queries and dynamic viewport units ($100dvh$) to ensure perfect layout rendering on both iOS Safari and desktop browsers.Interactive UI: Includes high-fidelity Lottie animations (confetti bursts) that trigger upon task completion to provide delightful user feedback.Theming & Styling: Implements a consistent design system using a custom ThemeProvider, featuring a soft color palette, responsive typography, and specialized global resets.Smart Date Formatting: Integrates Moment.js to display relative timestamps for when tasks were created.Tech StackFrontend: React (Vite)State Management: ZustandStyling: Styled ComponentsAnimations: Lottie-ReactUtilities: Moment.js
